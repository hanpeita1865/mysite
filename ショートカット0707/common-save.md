**Ctrl + S** は、作業中のファイルや文書の変更内容を保存するショートカットキーです。

##### 使い方

1. 文書やファイルを編集する
2. **Ctrl + S** を押す
3. 変更内容が保存される

##### 例

**文書を保存する場合**

```text
今日は良い天気です
```

文章を編集した後に **Ctrl + S** を押すと、変更内容が保存されます。

**ExcelやVSCodeで保存する場合**

* 表やコードを編集する
* **Ctrl + S** を押す

変更内容が保存されるため、作業内容を失うリスクを減らせます。

##### ポイント

* 作業中はこまめに保存する習慣をつけましょう。
* Windows、ブラウザの入力画面、VSCode、Excelなど、多くのアプリで共通して利用できます。
* 初めて保存するファイルでは、ファイル名や保存場所を指定する画面が表示されることがあります。
* 一部のアプリでは自動保存が有効でも、**Ctrl + S** を押すことで手動保存できます。

##### 覚え方

**S = Save（保存）**

頭文字そのままで覚えやすいショートカットです。

##### よくある組み合わせ

| 操作 | ショートカット |
|------|----------------|
| すべて選択 | Ctrl + A |
| コピー | Ctrl + C |
| 貼り付け | Ctrl + V |
| 元に戻す | Ctrl + Z |
| 保存 | Ctrl + S |

例えば、

1. Ctrl + A（すべて選択）
2. Ctrl + C（コピー）
3. 編集
4. Ctrl + S（保存）

と組み合わせることで、編集内容を安全に保存できます。
