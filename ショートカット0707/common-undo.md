Ctrl + Z は、直前に行った操作を取り消して、1つ前の状態に戻すショートカットキーです。

#### できること

* 文字入力を取り消す
* 削除した内容を元に戻す
* 移動や編集を取り消す
* 多くのアプリで共通して利用できる

#### 使い方

1. 誤って入力・削除・編集をする
2. Ctrl + Z を押す
3. 直前の操作が取り消される

#### 使用例

文章を元に戻す
```
今日は良い天気です
```
誤って「良い」を削除してしまった場合、Ctrl + Z を押すと削除前の状態に戻ります。

* Excel
: 入力した値や数式、削除したセルなどを元に戻せます。

* エクスプローラー
: 誤って削除したファイルや、変更したファイル名を元に戻せることがあります。

* ブラウザ   
: 入力欄で消してしまった文字を元に戻せることがあります。

* VSCode
: コードの編集や削除をすぐに取り消せます。

#### ポイント

* 「しまった！」と思ったらまず Ctrl + Z
* 多くのアプリで共通して使える
* 連続で押すと、さらに前の操作まで戻せる場合がある

#### 覚え方

Z = 時間を巻き戻すイメージ

失敗した操作を取り消して、少し前の状態へ戻します。

#### 利用できるアプリ

このショートカットは多くのアプリで共通して利用できます。

* Windows（エクスプローラー）
* ブラウザ
* VSCode
* Excel
* Word
* メモ帳 など

#### よくある組み合わせ

| 操作   | ショートカット  |
| ---- | -------- |
| 元に戻す | Ctrl + Z |
| やり直す | Ctrl + Y |
| コピー  | Ctrl + C |
| 切り取り | Ctrl + X |

#### 注意点

Ctrl + Z は非常に便利ですが、すべての操作を元に戻せるわけではありません。

特に Shift + Delete で削除した場合や、アプリを閉じた後などは元に戻せないことがあります。

そのため、誤って削除したときは まず Ctrl + Z を試す のがおすすめです。
