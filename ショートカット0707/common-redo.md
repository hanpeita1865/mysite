**Ctrl + Y** は、**Ctrl + Z** で取り消した操作をもう一度実行する（やり直す）ショートカットキーです。

#### 使い方

1. **Ctrl + Z** で操作を取り消す
2. **Ctrl + Y** を押す
3. 取り消した操作が再実行される

#### 使用例

**文章の編集をやり直す**

元の文章

```text
今日は良い天気です
```

「良い」を削除したあと、誤って **Ctrl + Z** を押して元に戻してしまいました。

その後 **Ctrl + Y** を押すと、再び削除が実行されます。

* Excel
: 元に戻したセルの入力や書式変更を、もう一度適用できます。

* エクスプローラー
: Ctrl + Z で取り消したファイルの移動や名前変更などを、再実行できる場合があります。

* ブラウザ
: 入力欄で **Ctrl + Z** により取り消した文字入力を、再度適用できることがあります。

* VSCode
: 元に戻したコード編集を、再び実行したいときに便利です。

#### ポイント

* **Ctrl + Z** とセットで使います。
* 「元に戻しすぎた」ときに便利です。
* 多くのアプリで利用できますが、一部のアプリでは **Ctrl + Shift + Z** が「やり直し」の場合もあります。

#### 覚え方

**Y = Yes（やっぱりその操作でよかった）**

Ctrl + Z で取り消した操作を、もう一度実行します。

#### 利用できるアプリ

このショートカットは多くのアプリで共通して利用できます。

- Windows（エクスプローラー）
- ブラウザ
- VSCode
- Excel
- Word
- メモ帳 など

#### よくある組み合わせ

| 操作 | ショートカット |
|------|---------------|
| 元に戻す | Ctrl + Z |
| やり直す | Ctrl + Y |
| コピー | Ctrl + C |
| 切り取り | Ctrl + X |

例えば、

1. 文字を削除
2. Ctrl + Z（元に戻す）
3. Ctrl + Y（やり直す）

で、取り消した操作をもう一度実行できます。

#### 注意点

**Ctrl + Y** は、**Ctrl + Z** で取り消した操作がある場合に利用できます。

また、別の操作を行ったあとや、アプリを閉じたあと、一部のアプリでは **Ctrl + Shift + Z** が「やり直し」に割り当てられている場合があります。
