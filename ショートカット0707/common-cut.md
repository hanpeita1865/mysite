**Ctrl + X** は、選択した文字やファイルなどを切り取り、別の場所へ移動するためのショートカットキーです。

#### 使い方

1. 切り取りたい文字・セル・ファイルなどを選択する
2. **Ctrl + X** を押す
3. 移動先で **Ctrl + V** を押す

#### 使用例

**文章を移動する**

```text
今日は良い天気です
```

「良い天気」を選択して **Ctrl + X** を押すと、その文字は一時的に取り除かれます。

その後、別の場所で **Ctrl + V** を押すと移動できます。

* Excel
: セルやセル範囲を選択して **Ctrl + X** を押すと、別の場所へ移動できます。

* エクスプローラー 
: ファイルやフォルダーを選択し **Ctrl + X** を押したあと、別のフォルダーで **Ctrl + V** を押すと移動できます。

* ブラウザ
: Webページ内の入力欄やフォームで選択した文字を切り取り、別の場所へ貼り付けられます。

* VSCode
: 選択したコードを別の場所や別のファイルへ移動するときに便利です。

#### ポイント

* **Ctrl + C** はコピー、**Ctrl + X** は移動です。
* **Ctrl + X** を押しただけでは完全には削除されず、クリップボードへ保存されます。
* **Ctrl + V** を押すと、切り取った内容を貼り付けられます。

#### 覚え方

**X = ハサミ（✂）の形**

文字やファイルを切り取るイメージで覚えると分かりやすいです。

#### 利用できるアプリ

このショートカットは多くのアプリで共通して利用できます。

- Windows（エクスプローラー）
- ブラウザ
- VSCode
- Excel
- Word
- メモ帳 など

#### よくある組み合わせ

| 操作 | ショートカット |
|------|---------------|
| 切り取り | Ctrl + X |
| 貼り付け | Ctrl + V |
| コピー | Ctrl + C |
| すべて選択 | Ctrl + A |

例えば、

1. Ctrl + A（すべて選択）
2. Ctrl + X（切り取り）
3. Ctrl + V（貼り付け）

で内容を別の場所へ簡単に移動できます。
