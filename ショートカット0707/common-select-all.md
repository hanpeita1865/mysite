**Ctrl + A** は、現在操作している画面や入力領域の内容をすべて選択するショートカットキーです。

#### 使い方

1. 文字・表・ファイルなどが表示されている画面を開く
2. **Ctrl + A** を押す
3. 現在の対象がすべて選択される

#### 使用例

**文章をすべて選択する**

文書内で **Ctrl + A** を押すと、すべての文字が選択されます。

その後、

* **Ctrl + C** でコピー
* **Delete** で削除
* そのまま入力して全置換

ができます。

* Excel 
: 表全体を選択して書式変更やコピーができます。

* エクスプローラー
: フォルダー内のファイルやフォルダーをすべて選択できます。

* ブラウザ 
: Webページ内の入力欄では、入力されている文字をすべて選択できます。

* VSCode
: 現在のエディター内のソースコード全体を選択できます。

#### ポイント

* ほとんどのWindowsアプリで共通して利用できます。
* コピー・切り取り・削除の前によく使用します。
* 選択後に文字を入力すると、選択範囲が置き換わるので注意してください。

#### 覚え方

**A = All（すべて）**

「全部選ぶ」と覚えると分かりやすいです。

#### よくある組み合わせ

| 操作 | ショートカット |
|------|---------------|
| すべて選択 | Ctrl + A |
| コピー | Ctrl + C |
| 切り取り | Ctrl + X |
| 貼り付け | Ctrl + V |
| 削除 | Delete |

例えば、

1. Ctrl + A
2. Ctrl + C

で現在の内容をすべてコピーできます。
