**Ctrl + C** は、選択した文字やファイルなどをコピーするショートカットキーです。

#### 使い方

1. コピーしたい文字・セル・ファイルなどを選択する
2. **Ctrl + C** を押す
3. 貼り付けたい場所で **Ctrl + V** を押す

#### 使用例

**文章をコピーする**

```text
今日は良い天気です
```

「良い天気」を選択して **Ctrl + C** を押し、メモ帳などで **Ctrl + V** を押すと貼り付けできます。

* Excel 
: セルやセル範囲を選択して **Ctrl + C** を押すと、値や数式、書式をコピーできます。

* エクスプローラー
: ファイルやフォルダーを選択し **Ctrl + C** を押したあと、別のフォルダーで **Ctrl + V** を押すと複製できます。

* ブラウザ
: Webページ内の入力欄やフォームで選択した文字をコピーできます。

* VSCode
: 選択したコードをコピーして、別の場所や別のファイルへ貼り付けられます。

#### ポイント

* コピーした内容はクリップボードへ保存されます。
* コピーしただけでは画面は変わりません。
* 新しい内容をコピーすると、以前の内容は通常上書きされます。
* 元の内容を移動したい場合は **Ctrl + X** を使用します。

#### 覚え方

**C = Copy（コピー）**

「C は Copy の C」と覚えると分かりやすいです。

#### 利用できるアプリ

このショートカットは多くのアプリで共通して利用できます。

- Windows（エクスプローラー）
- ブラウザ
- VSCode
- Excel
- Word
- メモ帳 など

#### よくある組み合わせ

| 操作 | ショートカット |
|------|---------------|
| コピー | Ctrl + C |
| 貼り付け | Ctrl + V |
| 切り取り（移動） | Ctrl + X |
| すべて選択 | Ctrl + A |

例えば、

1. Ctrl + A（すべて選択）
2. Ctrl + C（コピー）
3. Ctrl + V（貼り付け）

で内容を簡単に複製できます。
