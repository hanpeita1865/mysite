**Ctrl + V** は、コピーまたは切り取りした文字やファイルなどを貼り付けるショートカットキーです。

#### 使い方

1. **Ctrl + C**（コピー）または **Ctrl + X**（切り取り）を実行する
2. 貼り付けたい場所を選択する
3. **Ctrl + V** を押す

#### 使用例

**文章を貼り付ける**

```text
今日は良い天気です
```

「良い天気」を **Ctrl + C** でコピーし、メモ帳やメール本文で **Ctrl + V** を押すと貼り付けできます。

* Excel
: コピーまたは切り取ったセルやセル範囲を、別の場所へ貼り付けられます。

* エクスプローラー
: ファイルやフォルダーをコピーまたは切り取りし、別のフォルダーで **Ctrl + V** を押すと貼り付けられます。

* ブラウザ
: Webページ内の入力欄やフォームへ、コピーした文字を貼り付けられます。

* VSCode
: コピーまたは切り取ったコードを、同じファイルや別のファイルへ貼り付けられます。

#### ポイント

* **Ctrl + C** の後に使うとコピーした内容を貼り付けられます。
* **Ctrl + X** の後に使うと切り取った内容を移動できます。
* クリップボードに保存されている最新の内容が貼り付けられます。

#### 覚え方

**V = 挿し込む（Paste）のイメージ**

キーボード上で **C** の右隣にあり、「コピーしたらすぐ貼り付ける」と覚えると分かりやすいです。

#### 利用できるアプリ

このショートカットは多くのアプリで共通して利用できます。

- Windows（エクスプローラー）
- ブラウザ
- VSCode
- Excel
- Word
- メモ帳 など

#### よくある組み合わせ

| 操作 | ショートカット |
|------|---------------|
| 貼り付け | Ctrl + V |
| コピー | Ctrl + C |
| 切り取り | Ctrl + X |
| すべて選択 | Ctrl + A |

例えば、

1. Ctrl + C（コピー）
2. Ctrl + V（貼り付け）

で内容を簡単に複製できます。

また、

1. Ctrl + X（切り取り）
2. Ctrl + V（貼り付け）

で内容を別の場所へ移動できます。
